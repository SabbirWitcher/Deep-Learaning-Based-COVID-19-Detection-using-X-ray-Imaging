The goal of this project is to leverage the use of Deep Learning models to train a CNN classifier to classify COVID-19 or Tuberculosis from Normal Lungs using X-ray Images ensuring rapid detection of COVID-19 compared to the tradition RT-PCR techique which is time consuming. The project uses a simple GUI for the user to enter his X-ray image of a lung and then can get the result instantly.

The codes are used to train CNN models to classify different lung disease.
GUI is also developed for them for providing easier user interface.
